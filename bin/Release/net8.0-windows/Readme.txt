
explorerHelp - 0.2.1 - 2025/1/11 9:45

命令行参数说明：
  -v   显示版本信息
  -t   显示文件时间（可通过添加额外参数查看特定时间，如 /ct查看创建时间、/wt查看修改时间、/at查看访问时间）
  -a   查找文件位置
  -tt  修改文件时间（可通过添加 /fs 参数以时间戳方式修改文件最后修改时间，也可按指定日期时间格式修改文件创建、修改、访问时间）
  -d   删除文件（可添加 /r 参数进行强制删除，即永久删除）

使用方法示例：
  explorerHelp -v            # 显示程序版本信息
  explorerHelp -t <文件路径> [/ct|/wt|/at]  # 显示文件对应的创建、修改或访问时间
  explorerHelp -a <文件路径>  # 查找文件的位置
  explorerHelp -tt <文件路径> [/fs <时间戳或日期时间>] # 修改文件的时间戳或按指定时间修改文件时间
  explorerHelp -d <文件路径> [/r]  # 删除指定文件（可选择是否强制删除）
